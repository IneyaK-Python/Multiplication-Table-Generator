# Multiplication Table Generator
A beginner-friendly Python console application designed to help users practice basic arithmetic. The program takes a whole number input from the user, validates it to prevent crashes, and generates a structured 1-to-10 multiplication table until the user chooses to exit.

## Features
- *Input Validation:* Uses try/except logic to handle non-integer inputs gracefully without crashing.
- *Continuous Execution:* Runs in a controlled loop until the user types exit.
- *Clean Formatting:* Aligns columns using formatted string spacing for a readable console layout.

## How to Run
1. Ensure you have Python 3 installed on your system.
2. Open a terminal or IDE in the project directory.
3. Run the script using:
   bash
   python Table_Generator.py
   
4. Enter any whole number to view its table, or type exit to quit.
